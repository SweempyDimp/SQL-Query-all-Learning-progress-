WHAT IS DATABASE?
-- COLLECTION OF INFORMATION STORED IN AN ELECTRONIC FORMAT 

WHAT IS COLLECTION OF INFORMATION?
-- NO OF STUDENTS RECORDS/NO OF EMPLOYEE RECORDS/NO OF ORDERS INFORMATION
-- STUDENTS_DETAILS
-- STUDENT_ID  STUDENT_NAME  STUDENT_COURSE STUDENT_DOJ
-- 1101	  ABCD   DS   01-JAN-2025
-- 1102   EFGH   SQL  01-FEB-2025

THIS COLLECTION OF INFORMATION IS REQUIRED TO STORED IN AN ELECTRONIC FORMAT
-- EX: 
STORE COLLECTION OF INFORM IN EXCEL FILE/NOTEPAD/CSV -- DATABASE

DATABASE ACTIVITIES 
-- MODIFY EXISTING INFORMATION
-- DELETE EXISTING INFORMATION
-- CREATE NEW INFORMATION

TO PERFORM DATABASE ACTIVITIES -- SOFTWARE IS REQUIRED 
DBMS -- DATABASE MANAGEMENT SYSTEM 
-- SOFTWARE USED TO PERFORM DIFFERENT ACTIVITIES ON THE DATABASE
EX: EXCEL -- STORING DATA /PERFORM DIFFERENT ACTIVITIES ON THE DAT STORED

WHAT IS RELATIONAL DATABASES?
-- DATABASE THAT STORES INFORMATION IN THE FORMAT OF TABLES 
-- DATABASE THAT SUPPORTS ALL RELATIONSHIPS BETWEEN TABLES

WHAT IS RDBMS?
-- RELATIONAL DATABASE MANAGEMENT SYSTEM
-- SOFTWARE IS USED TO MANAGE DATABASE ACTIVITIES IN THE RELATION DATABASE

EX: MS ACCESS, MS SQL SERVER, ORACLE, MYSQL, POSTGRESQL..

WHAT ARE RELATIONSHIPS IN RELATIONAL DATABASES?
1:1, 1:M, M:1, M:M

WHAT IS SQL?
SQL - STRUCTURED QUERY LANGUAGE

IS SQL A DATABASE?
NO
IS A PROGRAMMING LANGUAGE USED TO INTERACT WITH RELATIONAL DATABASES

WHAT IS DIFFERENCE BETWEEN SQL & MS SQL SERVER?
SQL -- PROGRAMMING LANGUAGE 
MS SQL SERVER -- DATABASE STORAGE

-- AN INFORMATION THAT CAN BE ORGANISED STORED IN THE FORMAT OF TABLES RELATIONAL DATABASE

---- UNSTRUCTURED DATA 
---- NON RELATIONAL DATABASES
-- AN INFORMATION THAT CANNOT BE ORGANISED & CANNOT STORED IN TABLE FORMATS
-- GRAPHS, IMAGES, VIDEOS, EMAILS, POSTS, CHAT MESSAGES, TEXT MESSAGES, XML, JSON...

EX: MONGODB  -- AN INFORMATION OF TYPE KEY/VALUE PAIR FORMAT 
EG JSON LIKE DOCUMENTS,
EX: NEO4J -- GRAPH DATABASE -- NODES AND EDGES 
MONGODB, CASSANDRA, REDIS, NEO4J

-- MICROSOFT SQL SERVER DATABASE [MS SQL SERVER DATABASE]
TO WORK WITH SQL SERVER DATABASE 2 APPLICATIONS REQUIRED
-- SQL SERVER DATABASE [DEVELOPER/EXPRESS/ENTERPRISE]
ACTUAL DATABASE WHERE INFORMATION IS STORED

-- SQL SERVER MANAGEMENT STUDIO [SSMS]
RDBMS SOFTWARE TAKES SQL COMMANDS FROM USERS & SENT TO DATABASE SERVER
NO DATA STORAGE 

-- SQL COMMANDS
-- ARE USED TO INTERACT WITH RELATIONAL DATABASE

SQL COMMANDS ARE CATEGORISED AS BELOW:

DDL -- DATA DEFINITION LANGUAGE
COMMANDS RELATED TO TABLE STRUCTURE
CREATE/ALTER/DROP

DML -- DATA MANIPULATION LANGUAGE
COMMANDS RELATED TO DATA 
INSERT/UPDATE/DELETE

TCL -- TRANSACTION CONTROL LANGUAGE
COMMANDS RELATED TO TRANSACTION [A TRANSACTION IS INSERT/UPDATE/DELETE]
SAVE/COMMIT/ROLLBACK

DQL -- DATA QUERY LANGUAGE
SELECT -- COMMAND USED TO RETRIEVE DATA PRESENT IN A TABLE
